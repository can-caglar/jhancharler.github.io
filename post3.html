Windows Software for MARS

The software above was written using the .NET framework (C#). Its function is to provide a platform for consumers to easily set up their Mars system and play lightgun games on the PC.

I was in charge of the design and development of this software.

This task required me to learn to create applications for Windows, Windows kernel drivers, signing kernel drivers, create robust and easy-to-use user interfaces.

Software features:

  • Able to calibrate and read data from a Mars lightcon and Mars IR-Station. The data is translated to 'rawinput' mouse data (i.e. it makes Mars work like a USB mouse).
  • Works with up to 4 players.
  • Able to change the colour of Mars lightcon LEDs for each of the players.
  • Driver installer package which installs all the necessary drivers via a helpful installer wizard. The drivers are all signed, so no scary warning messages!
  • Colour-coded, flashing instructions to help users easily understand how to get set up.
  • An instructions window which displays additional tips.
  • Universal hotkeys (hotkeys which work regardless of which application is in focus). This helps with things such as stopping the gun tracking or re-calibration without needing to switch focus to this application.
  • Ability to re-configure hotkeys.
  • Automatically save configurations on exit.

Skills developed:

  1. Writing Windows Forms applications with C#.
  2. Knowledge on improving user experience with the user interface and application features.
  3. C# bug fixing using the Visual Studio IDE.
  4. Windows driver development.
  5. Windows driver signing.

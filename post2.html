My University Projects at Leeds

iPhone App

I have posted pictures above of a couple of my university projects.

The first set of photos shows the iPhone application I developed during my second year called 'Painless Binary'. This project taught me how to design intelligent UI and write iPhone apps in Objective-C. The purpose of the app was to allow users to easily convert between binary, hexadecimal and decimal numbers in one screen.

Nokia Style Gaming Device

The second set of photos is of my Nokia style embedded gaming device. This project included the following:

  1. Design a standalone circuit using an ATMEGA328P microcontroller which allows it to be programmed and interfaced with an LCD and input peripherals.
  2. Design and develop a fully functional game.
  3. Create a schematic of the design.
  4. Create a PCBA based off the schematic.
  5. Solder the components on the PCBA and debug.

The final product was a working game ‘Escape the Darkness’ whereby you control a character out of a maze without hitting the borders, whilst trying to remember the exit points as the room fills with darkness from below!
